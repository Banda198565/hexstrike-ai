# HexStrike On-Chain Forensics — Правила оператора

## Путь запуска
- Все 3 прогона: `bash scripts/run-three-progons.sh`
- Forensics-модули: `source scripts/forensics-env-mac.sh && ./scripts/run-all-forensics.sh`
- Тест 5 кошельков: `python3 scripts/hexstrike-orchestrator.py run field-targets-5`
- Аудит оператора: `python3 scripts/hexstrike-orchestrator.py run operator-lab`
- Комbined workflow: `python3 scripts/hexstrike-orchestrator.py run operator-targets-3progon`
